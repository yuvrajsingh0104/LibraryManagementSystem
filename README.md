# Library Management System

## Project Overview

This Library Management System is designed to help libraries manage their books, patrons, and lending processes efficiently. It features an intuitive user interface developed in Java and robust data handling capabilities using SQL databases. The system simplifies the management of library catalogs and enhances the user experience for both library staff and patrons.

## Features

- **Catalog Management**: Add, update, and remove books from the library's catalog.
- **Membership Management**: Manage patron information and membership details.
- **Borrowing and Returning**: Handle the checkout and return processes for books.
- **Search Functionality**: Allows users to search for books by various criteria, including title, author, or ISBN.
- **Due Dates and Fines**: Automatically calculate due dates and fines for overdue books.

## Technology Stack

- **User Interface**: Java Swing
- **Backend Logic**: Java SE
- **Database**: MySQL
- **Development Environment**:
  - JDK 17
  - MySQL Server 8.0
  - Eclipse or any Java IDE

### Prerequisites

- Java JDK 17 or later
- MySQL Server 8.0 or later
- Java IDE (Eclipse, IntelliJ IDEA, or similar)
